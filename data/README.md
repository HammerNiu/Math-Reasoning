# Data Artifacts

Generated trajectory and evaluation files are written here.

Common files:

- `trajectories.jsonl`: output from `tools/collect_trajectories.py`
- `experiment_results.json`: output from `experiments/run_experiment.py`
- `eval_results.json`: output from `eval.py`

These files are generated by API-backed runs and may be omitted from the git
history if they contain large outputs or model responses. The commands in the
top-level README reproduce them.
