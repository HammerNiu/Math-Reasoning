Before submission:

1. Confirm the author, UNI, and demo-video link fields.
2. Compile in the ACM SIGPLAN/ACM conference template environment, for example
   by uploading `final_report.tex` to Overleaf's ACM template project.
3. Export the PDF and name it `hn2477_sy3302_rw3152.pdf`, matching the submission
   guidance.
4. Check that the GitHub repository link and demo-video link open in an
   incognito/private browser.

The local machine currently lacks `acmart.cls`, so `final_report.tex` includes a
two-column fallback only for local compile checks. On Overleaf, or on a TeX
installation with `acmart`, the same source will use the required ACM class.
